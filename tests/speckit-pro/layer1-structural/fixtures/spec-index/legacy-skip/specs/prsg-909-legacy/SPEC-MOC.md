# PRSG-909 — Map of Content (legacy, NOT version-marked)

This note carries NO `structureVersion`, so `moc_is_gated` returns false and the
generator skips it entirely (FR-007). It must be left byte-for-byte unmodified by
any run — no zone injection, no zone write.
