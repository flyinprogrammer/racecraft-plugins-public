Racecraft Public Plugins is the public marketplace for Racecraft-maintained agent plugins. The current plugin is `speckit-pro`, which gives teams a structured SpecKit workflow for moving from specifications to reviewable implementation work in Claude Code and Codex.

Choose the platform you use:

- [Install speckit-pro for Claude Code](install/claude-code/) if your workflow runs through Claude Code plugin commands.
- [Install speckit-pro for Codex](install/codex/) if your workflow runs through Codex skills, agents, and plugin surfaces.

## Route Scope

- **Audience:** First-time user
- **Purpose:** Explain Racecraft Public Plugins, speckit-pro, platform choice, and first next step.
- **Shell owner DOC:** DOC-002
- **Full-content owner DOC:** DOC-002
- **Success criterion:** User chooses Claude Code or Codex path within one screen.

## What You Get

`speckit-pro` helps keep specification, planning, autopilot execution, review, and status workflows connected to the same project evidence. Start with the platform path that matches your agent environment.

## Source And Payloads

The authoring source lives in `speckit-pro/`. Installable payloads are generated under `dist/claude/**` and `dist/codex/**` for their target platforms, so generated package layout can differ from the source tree.

[Review source and generated payloads](reference/) before comparing files or reporting packaging issues.
